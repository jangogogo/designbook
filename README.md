# designbook

游戏开发基础 skill 仓库。

本仓库把两类记忆整理成一个可复用 Codex skill:

- `ParanoiaSkills` 的游戏设计生产方法: 证据优先、验证契约、概念到原型、经验密度、可回滚工作流。
- 本机多个游戏项目的工程共性: Godot C#、逻辑/表现分离、数据驱动、内置编辑器、固定 seed trace、薄切片验证。

## Skill

- `skills/game-dev-foundation`

适用场景:

- 新建 Godot/C# 游戏项目。
- 审查现有游戏工程架构。
- 把玩法想法落成可验证原型。
- 设计数据驱动战斗、Buff、AI、关卡、编辑器和测试闭环。
- 迁移 Codex 时恢复游戏开发协作习惯。
